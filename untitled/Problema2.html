Neste problema é fornecida uma fórmula que dá o número de Catalan e é pedido que, a patir dessa fórmula, se derive uma implementação da mesma mas que não calcule fatoriais. Depois dessa implementação derivada, é pedido que seja construído o ciclo-for que implemente essa função.

Derivamos então uma implementação recursiva de Cn sem fatoriais, através da definição do triângulo de Pascal. Apresentamos de seguida os cálculos efetuados:
Ficamos, então com o seguinte: Cn+1 = (4n+2)/(n+2) * Cn.

A partir daqui, foi necessário criar recursividade mútua, para ser possível definir o ciclo-for.

Foram definidas duas funções, uma que calcula o dividendo da divisão, e outra que calcula o divisor.

Para as definir, baseamo-nos num exemplo do caderno da disciplina (pag. 112) em que a função ns é definida com recursividade mútua.

A função dividendo deve calcular a seguinte expressão, dependente do valor de n: 4n+2. Como caso base temos que se n = 0, então 4*0 + 2 = 2, para os restantes casos, o objetivo será adicionar 4, n vezes, então, ficaremos com a seguinte definição:

  • dividendo 0 = 2
  • dividendo (n+1) = 4 + dividendo n

A função divisor deve calcular a expressão n+2, dependendo do valor de n. Como caso base temos então que se n = 0, n+2 = 2, para os restantes, o raciocínio é o mesmo que na função anterior, ficamos então com:

  • divisor 0 = 2
  • divisor (n+1) = 1 + divisor n
